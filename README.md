# Eeveedistro

Eeveedistro is a small custom ROM designed for transferring Pokémon to Gen 1 and 2 (through time capsule) Pokémon games. At this moment, it's capable of transferring a PCNY Growth Eevee with maxed DVs.

This project can be easily modified to transfer any Pokémon.

## Building
You need **gbdk-2020** to build the project (version 4.20 was used at the time of writing this but I guess newer ones will also work). Then simply run `make`.
